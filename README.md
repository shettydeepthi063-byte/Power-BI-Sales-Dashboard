# 📊 Sales Performance Dashboard using Power BI
 
## 🔹 Overview
Developed an interactive Power BI dashboard to analyze sales performance across different regions and product sub-categories. The dashboard enables quick insights into revenue distribution and trends.
 
## 🔹 Dashboard Preview

 <img width="1221" height="560" alt="Sales_Dashboard_PowerBI" src="https://github.com/user-attachments/assets/c80854f2-d346-4e55-9b77-0ebf2f1703fe" />
 
## 🔹 Features
- Sales Distribution by Region (Donut Chart)
- Sub-category Sales Analysis (Funnel Chart)
- Sales Trend Over Time (Line Chart)
- KPI Card displaying Total Sales
 
## 🔹 Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Basic Aggregations)
 
## 🔹 Dataset
- Sample sales dataset used for analysis and visualization
 
## 🔹 Key Insights
- West region contributes the highest share of total sales
- Certain sub-categories drive the majority of revenue
- Sales trends show variation over time
